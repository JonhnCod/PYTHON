Desenvolvimento do Site Responsivo: Uma Jornada Desafiadora


Olá, sou Jonathan, e quero compartilhar a experiência de criar meu site responsivo. Desde a padronização visual até a implementação de menus persistentes e organização estruturada, o projeto tem sido desafiador e gratificante.


Padronização Visual:


Optei por um design uniforme em todas as páginas, cuidando das cores, fontes e imagens para uma identidade visual consistente.


Menu Onipresente:


Incluí um menu persistente para facilitar a navegação do usuário, presente em todas as páginas.


Estrutura Organizada:


Seguindo o roteiro, criei os arquivos principais e organizei pastas para scripts, imagens e estilos, mantendo tudo claro e organizado.


CSS Eficiente:

Utilizei regras de estilo em classes para formatação visual eficiente, facilitando a manutenção e atualização futura.


Javascript Dinâmico:

Integrei Javascript para dinamizar a interação do usuário, destacando a criação de um formulário com validação abrangente.


Caminhos Relativos:

Adotei caminhos relativos para garantir a portabilidade do site, independente do local de hospedagem.


Em resumo, esse projeto, embora desafiador, reflete meu comprometimento em proporcionar uma experiência envolvente aos usuários. Fique atento para o vídeo, onde detalharei cada aspecto do desenvolvimento.
